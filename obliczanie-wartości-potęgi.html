Opis Projektu

Kalkulator Potęgowania to narzędzie umożliwiające szybkie obliczenia związane z podnoszeniem do potęgi. Wprowadź bazę i wykładnik, a kalkulator zwróci wynik potęgowania, dzięki czemu łatwo możesz uzyskać wartość potęgi dla dowolnych liczb.

Kod Źródłowy

                #include 
                #include 
                
                int main() {
                    // Wprowadź podstawę i wykładnik
                    double base, exponent;
                    std::cout << "Podaj podstawe i wykladnik (oddzielone spacja): ";
                    std::cin >> base >> exponent;
                
                    // Oblicz potęgę
                    double power_result = std::pow(base, exponent);
                    std::cout << base << " do potegi " << exponent << " to: " << power_result << std::endl;
                
                    return 0;
                }
