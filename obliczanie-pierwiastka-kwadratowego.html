Opis Projektu

Kalkulator Pierwiastka Kwadratowego to prosty narzędzie, które umożliwia obliczenia związane z równaniami kwadratowymi. Dzięki niemu można szybko i łatwo uzyskać wartości pierwiastków oraz delta dla podanych współczynników a, b i c.

Kod Źródłowy

                #include 
                #include 
                
                int main() {
                    // Wprowadź liczbę
                    double x;
                    std::cout << "Podaj liczbe: ";
                    std::cin >> x;
                
                    // Oblicz pierwiastek kwadratowy
                    double sqrt_result = std::sqrt(x);
                    std::cout << "Pierwiastek kwadratowy z " << x << " to: " << sqrt_result << std::endl;
                
                    return 0;
                }
